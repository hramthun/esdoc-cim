An interaction between the use of a property by a process in a particular way (a property role) and the use of the same property by another process in a particular way (another property role).  This defines the connections between inputs and outputs for a pair of processes.<br /></p><p><br /></p><p>Instances of the ProcessInteraction class could be derived from the PropertyConnection class and the former would not be required, except that for some processes, property connections are not known or not specified, so the interactions between two processes must be described at the process level.<br /></p>
